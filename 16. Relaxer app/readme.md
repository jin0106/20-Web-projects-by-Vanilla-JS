## 2. 프로젝트를 통해 배운것

#### 1) conic-gradient() 

CSS기능으로 색상 변화와 그라데이션 - 중심점을 기준으로 회전하여 만들어준다.

```css
background: conic-gradient(#55b7a4 0%,
      #4ca493 40%,
      #fff 40%,
      #fff 60%,
      #336d62 60%,
      #2a5b52 100%);
```
